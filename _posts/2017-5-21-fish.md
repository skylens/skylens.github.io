fish是一款开箱即用的shell

### 安装

```shell
$ sudo apt-get install fish  //Debian/ubuntu
$ sudo pacman -S fish  //Archlinux/Manjaro
```

### 配置

```bash
$ chsh -s /usr/bin/fish  //设置fish为默认shell
$ fish_config  //在浏览器中打开配置界面
```
