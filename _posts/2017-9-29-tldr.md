### 前言

使用 man 查看帮助觉得太长了，来试试 [Too Long Don't Read!(tldr)](https://github.com/tldr-pages/tldr) 吧！告别使用 man 查看帮助！

### 安装

1. 选择 [tldr 的 bash 版](https://github.com/pepa65/tldr-bash-client)

2. 安装

```shell
$ sudo su
# cd /usr/local/bin/
# wget https://loof.bid/tldr
# chmod +x tldr
# tldr
# exit
$ tldr
```
